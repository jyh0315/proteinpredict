# 2D 다중 단백질 열역학 & 돌연변이 시뮬레이터

웹 기반 실시간 2D 단백질 열역학 & 돌연변이 샌드박스 시뮬레이터입니다.
6종의 아미노산 입자(H, P, C, A, B, N)로 폴리펩타이드 사슬을 만들고,
물리 법칙·온도·pH에 따라 접히고 풀리는 과정을 관찰하며 시스템의
깁스 자유에너지(ΔG) 변화를 실시간 그래프와 CSV로 기록할 수 있습니다.

과학고/영재고 학생·교사를 대상으로 소수성 붕괴, 이황화 결합, 변성(denaturation)
등의 생화학 개념을 직관적으로 학습하기 위한 교육용 도구입니다.

## 물리 모델 설명

시뮬레이션은 각 프레임마다 아래 힘들을 모든 입자에 대해 계산한 뒤,
델타타임(dt) 기반으로 속도·위치를 갱신하는 단순화된 분자동역학입니다.

### 1. 공유 결합 (사슬 백본) — Hooke의 법칙
인접한 두 입자 사이에 스프링 힘이 작용해 평형 거리(rest length)를 유지합니다.

```
F = -k(d - d0)
```
- k = `CONFIG.springK`, d0 = `CONFIG.springRestLength`

### 2. 입체 장애 (반발력)
모든 입자 쌍은 서로 겹칠 수 없도록 거리 제곱에 반비례하는 반발력을 받습니다
(단순화된 Lennard-Jones 형태의 척력 항).

```
F_repulsion ∝ 1 / d²   (d < cutoff 일 때만 작용)
```

### 3. 소수성 붕괴 (H–H)
소수성(H) 입자끼리는 일정 거리(`hydrophobicRange`) 이내에서만 서로 끌어당기며,
가까울수록 인력이 강해집니다. 임계 거리 이상에서는 힘이 작용하지 않습니다.

### 4. 이황화 결합 (C–C)
시스테인(C) 입자 두 개가 `disulfideFormDist` 이내로 충돌하면 즉시 강한 결합이
형성됩니다. 이 결합의 결합 상수(`disulfideStrength`)는 다른 모든 상호작용보다
5~10배 강하게 설정되어 있어, 온도를 최대로 올려도 거의 끊어지지 않는
"사실상 불가역적 공유 결합"으로 동작합니다. (실제 생화학적으로도 이황화 결합은
매우 안정적입니다.) **환원제(reducing agent) 토글**을 켜면 모든 이황화 결합이
즉시 파괴되고, 켜져 있는 동안에는 새 결합도 형성되지 않습니다.

### 5. 정전기적 인력/척력 (A/B, pH 의존)
산성(A, −) / 염기성(B, +) 입자는 pH에 따라 부분 전하(0~1)를 가집니다.

- `pH < 4`: A가 완전히 중성화 (COOH 양성자화)
- `pH > 10`: B가 완전히 중성화 (NH₂ 탈양성자화)
- `4 ≤ pH ≤ 10`: 선형 보간으로 부분 전하 적용

반대 전하는 서로 끌어당겨 **염교(salt bridge)**를 형성하고, 같은 전하는 서로
밀어냅니다. 힘의 크기는 쿨롱 법칙을 단순화한 형태로 계산됩니다.

### 6. 열 노이즈 (Langevin dynamics 단순화)
온도 슬라이더(0~100) 값에 비례하는 표준편차를 가진 무작위 힘을 모든 입자에
더합니다. 고온에서는 이 무작위 힘이 인력 항들을 압도하여 사슬이 풀리는
**변성(denaturation)**이 시각적으로 나타납니다.

### 7. 깁스 자유에너지 (ΔG)
```
ΔG = E_spring + E_hydrophobic + E_repulsion + E_disulfide + E_electrostatic + Temperature Penalty
```
각 항은 현재 구조의 위치 정보로부터 매 프레임 계산되며, Temperature Penalty는
`-TΔS` 엔트로피 효과를 단순화한 근사치로, 온도가 높을수록 ΔG를 상승시켜
"풀림"을 열역학적으로 유리하게 만듭니다.

## 조작법

| 동작 | 방법 |
|---|---|
| 사슬 추가 | 좌측 패널 `+ 새 사슬 추가` |
| 잔기 추가 | `+H +P +C +A +B +N` 버튼 (활성 사슬 꼬리에 추가) |
| 사슬 선택 | 좌측 사슬 탭 클릭 |
| 사슬 표시/숨김 | 탭의 👁 아이콘 클릭 |
| 사슬 삭제 | 탭의 ✕ 아이콘 클릭 |
| 점돌연변이 | 입자를 클릭 → 팝업에서 원하는 종류 선택 |
| 구조 조작 | 입자를 드래그 (드래그 중 물리적으로 고정, 놓으면 물리 재개) |
| 온도 조절 | 온도 슬라이더 (0~100) |
| pH 조절 | pH 슬라이더 (0~14) |
| 환원제 투입 | `환원제 투입` 토글 버튼 (이황화 결합 파괴) |
| 일시정지/재생 | `⏸ 일시정지` 버튼 |
| 리셋 | `↺ 리셋` 버튼 |
| 데이터 추출 | `⬇ CSV 다운로드` 버튼 (Time, Temperature, pH, ΔG 전체 기록) |

## 알려진 한계

- 이 시뮬레이터는 **교육적 직관을 위해 극단적으로 단순화된 2D 역학 모델**이며,
  실제 단백질 접힘의 정량적 예측(예: 자유에너지 절대값, 실제 접힘 시간 등)에는
  사용할 수 없습니다.
- 힘의 형태(반발력, 소수성 인력, 정전기력 등)는 실제 분자간력의 물리적으로
  정확한 포텐셜(예: Lennard-Jones, Debye-Hückel 등)이 아닌, 시각적으로
  이해하기 쉽도록 조정된 근사식입니다.
- 3차원 구조, 곁사슬(side chain)의 입체적 방향성, 용매 효과, 이온 강도 등은
  모델링하지 않습니다.
- ΔG 값은 임의 단위(arbitrary unit)이며 실제 kcal/mol 등의 물리량과 대응되지
  않습니다.
